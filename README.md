
# FoodPlaner 

FoodPlaner is a website who can create a menu for the whole week with a simple button and a beautifull ui !


## Deployment

To run this website :

```bash
  cd src/python
```
```bash
  source venv/bin/activate
```
```bash
  python3 main.py
```
```bash
  npm run dev
```
